# SDK Wrapper Side Path

The SDK wrapper path was explored as image-format research.

It is not the mainline OpenWrt port, and it must not drive project decisions.
The primary project path is clean-room support for ipTIME A2004MU in a
latest/official-style OpenWrt tree.

The wrapper tools may remain as reference tooling for understanding observed
header fields, checksums, payload offsets, and layout constraints. They are not
a substitute for clean-room OpenWrt target, DTS, driver, image recipe, or board
default work.

Experimental candidates generated by this side path are not flash-verified.
They must not be treated as final firmware or as proof that mainline OpenWrt
support exists.
